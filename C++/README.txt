Caesar/Date Cipher:

Build with make all to get both programs.
Implements a caesar cipher on a hardcoded string. Can be modified to accept input. Also has a date cipher that scrambles dates.

Maze Solver:

Build with make all.
Takes a text input representation of a maze and finds a solution to traverse it, while presenting the user with updates on each decision, prompted by the enter key.

Sentence Parser:

Build with make all.
Parses strings and can determine word count, phrase count, and other statistics about paragraphs and sentences using maps and lambda expressions.
